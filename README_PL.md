# Peru 2026 — GitHub Pages

## Najprostsza instalacja bez terminala

1. Rozpakuj plik ZIP na komputerze.
2. Załóż konto na github.com.
3. Kliknij znak `+` w prawym górnym rogu → **New repository**.
4. Nazwa repozytorium: `peru-2026`.
5. Ustaw **Public** i kliknij **Create repository**.
6. W repozytorium kliknij **Add file → Upload files**.
7. Przeciągnij do okna CAŁĄ ZAWARTOŚĆ rozpakowanego folderu:
   - index.html
   - styles.css
   - app.js
   - manifest.webmanifest
   - service-worker.js
   - .nojekyll
   - folder icons
8. Na dole kliknij **Commit changes**.
9. Wejdź w **Settings → Pages**.
10. Source: **Deploy from a branch**.
11. Branch: **main**, folder: **/(root)**.
12. Kliknij **Save**.

Po chwili strona będzie dostępna pod adresem:
`https://TWOJA-NAZWA.github.io/peru-2026/`

## Ważne

Nie dodawaj do publicznej strony:
- kodów rezerwacji,
- numerów dokumentów,
- biletów PDF,
- prywatnych adresów i telefonów.

Do tego służy prywatny Notion.
