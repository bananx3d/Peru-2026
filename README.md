# Peru 2026 — Expedition Field Guide v2

Nowa wersja usuwa grafiki AI i używa prawdziwych fotografii z oficjalnej strony Curassow Lodge oraz Wikimedia Commons.

## Publikacja
1. W lokalnym folderze repo usuń starą zawartość (nie usuwaj folderu `.git`).
2. Wklej całą zawartość tego folderu.
3. GitHub Desktop: `Commit to main` → `Push origin`.
4. GitHub Pages pozostaje na `main / (root)`.

## Offline
Interfejs i tekst są zapisane od razu. Prawdziwe fotografie z zewnętrznych źródeł są zapisywane w cache po pierwszym wyświetleniu. Otwórz wszystkie ważne rozdziały raz przy dobrym Wi‑Fi przed wyjazdem.

## Prywatność
W aplikacji nie ma kodów rezerwacji, prywatnych adresów, telefonu Martina ani salda. Dane prywatne są w sejfie Notion.
